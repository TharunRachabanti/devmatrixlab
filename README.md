# DevMatrix

DevMatrix is a comprehensive web application built with [React](https://react.dev/) and [Vite](https://vitejs.dev/), designed to provide a seamless and interactive user experience. It features a modern design with smooth animations, responsive layouts, and a variety of service-oriented pages.

- **Modern Tech Stack**: Built with React 19 and Vite for lightning-fast development and build performance.
- **Responsive Design**: Fully responsive layout ensuring compatibility across desktops, tablets, and mobile devices.
- **Smooth Animations**:
  - **AOS (Animate On Scroll)**: Elements fade and slide in as you scroll.
  - **Framer Motion**: Complex, physics-based animations for interactive elements.
- **Interactive Carousel**: Utilizes **Swiper** for touch-friendly sliding content.
- **Client-Side Routing**: Seamless navigation using **React Router v7**.
- **Smooth Scrolling**: Enhanced navigation experience with **react-scroll**.
